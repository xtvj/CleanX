## 项目目的：

- 练手：尽可能使用Jetpack组件。

- 自用APP：速度快、功能简单、不附带杂七杂八的功能、不频繁升级。

# 

## 技术点：

- Coroutines
- Hilt
- ViewModel
- ViewBinding
- Startup
- Lifecycle
- Flow
- Room
- Paging 3
- DataStore
- Workmanager
